# Galactic Arena
Galactic Arena is an intense, fast-paced top-down shooter where your sole objective is survival. Set in a space complex, you must face relentless waves of enemies, testing your reflexes and strategy. The game, developed with GameMaker, offers endless action, high scores to beat, and a casual yet thrilling 2D experience.
